Instruktioner:
1. Programmet skall startas från main.py
2. Kommentarer
Kommentera er kod med # och tydliga namn.
Använd type hinting för att visa mig vad en variabel är tänkt att innehålla.
Kommentera klasser och funktioner med en fler-rads-sträng. Dessa syns sedan om man använder help()
3. Format:
Variabler döps med snake_case eller camelCase.
Variabler börjar med liten bokstav
Klasser börjar med stor bokstav
Tydliga och förklarande namn, dvs. läslig kod
Hellre enkelt än komplicerat

4. Funktioner

Skapa funktion(er) för kod som annars skulle upprepas
Använd type hinting for att visa vad funktioner returnerar. Tips func() -> datatyp:
5. Klasser
Ert program skall innehålla minst en egen class. 
Klasser skall ligga i egna .py-filer som hämtas in med import
6. Moduler
Ni skall använda minst en modul från Python Standard Library. https://docs.python.org/3/library/index.html
Helst inte använda moduler som kräver pip install, men undantag ges för “requests” (om ni vill hämta data från nätet) https://pypi.org/project/requests/
Vill ni använda andra Pypi.org moduler, så kolla med läraren först 
7. Filer
Er kod skall spara data i en (eller flera) fil(er), lättast med text i rader, men JSON eller annat format är välkommet.
Koden skall även kunna läsa sagda fil(er)
8. Villkor
Ni skall ha med villkor i koden. If-satser är ett krav, match-case är valfritt. Gärna båda.
9. Flöde
Programmet skall kunna göra flera saker, t.ex lägga till, ta bort etc.
10. Generellt
main.py skall ha en meny för att visa valen i flödet
